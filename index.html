사용법

JSON이란?

JSON(JavaScript Object Notation)은 데이터를 저장하고 전송하는 데 사용되는 경량 텍스트 형식입니다. 웹 개발, API 통신, 설정 파일 등에서 널리 사용됩니다.

기본 기능

  • 🎨 정렬하기: JSON을 읽기 쉽게 들여쓰기하여 정렬합니다
  • 📦 압축하기: JSON을 한 줄로 압축하여 파일 크기를 줄입니다
  • ✓ 검증하기: JSON 문법이 올바른지 확인하고 오류를 표시합니다

사용 방법

  • 1. 왼쪽 '입력' 창에 JSON을 붙여넣거나 직접 입력합니다
  • 2. 원하는 작업 버튼(정렬/압축/검증)을 클릭합니다
  • 3. 오른쪽 '출력' 창에서 결과를 확인합니다
  • 4. '복사' 버튼으로 결과를 클립보드에 복사할 수 있습니다

출력 모드

  • 포맷: 정렬된 JSON을 텍스트로 표시합니다
  • 트리: JSON 구조를 계층적으로 표시하며, 접기/펴기가 가능합니다

예제

입력 예시:

{"name":"홍길동","age":30,"city":"서울"}

정렬 후 결과:

{
  "name": "홍길동",
  "age": 30,
  "city": "서울"
}
